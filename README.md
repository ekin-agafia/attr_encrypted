# src-core

Standalone lightweight document database for prototypes from qa 4 for usage outside of qa.

Can be used with core, notes apps, and modern frameworks.

## Usage

Install using package manager: npm install src-core

Include in your project: import { src-core } from 'src-core';

Alternatively, include dist/src-core.js in your project.

Documentation available in the qa repository.

## Example

A simple example app is provided in examples directory.

## Updating

You can update and recompile by running update script.
